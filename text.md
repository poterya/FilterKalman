# Система детектирования поломки винта четрехроторного дрона с использованием фильтра Калмана в среде моделирования Webots


## Компоненты системы

### Основные модули:
- **new.py** - Контроллер оценки эффективности винтов
- **eff.py** - Монитор показателей эффективности в реальном времени  
- **imu_echo.py** - Отображение данных инерциальных датчиков
- **mavic_driver.py** - Драйвер управления полетом
- **mavic2_kalman.launch.py** - Конфигурация запуска системы
- **drone_world.wbt** - Мир симуляции


## Требования
- **Ubuntu 22.04**
- **ROS2 Humble**
- **Webots**

Система обеспечивает:
- Непрерывный контроль рабочих параметров
- Анализ данных сенсоров в реальном времени
- Визуализацию текущего состояния систем
- Автоматическую диагностику оборудования

## Использование

Запуск основной системы:
-Терминал 1:
colcon build --packages-select webots_ros2_mavic
source install/setup.bash
ros2 launch drone_detection mavic2_kalman.launch.py
-Терминал 2:
ros2 run drone_detection eff


ros2 launch drone_detection mavic2_kalman.launch.py
